Features

  • 480x480 RGB display with LVGL graphics library
  • Colored squares display test
  • Improv Wi-Fi support for easy WiFi configuration
  • ESP32-S3 with PSRAM support
  • Web-based flashing (no IDE required)

ðŸ’» Compatible Hardware

This firmware is designed for the Guition ESP32-S3-4848S040 development board with a built-in 480x480 ST7701S display. Other ESP32-S3 boards with similar display configurations may also work.

Install Firmware

ðŸ“‹ Installation Instructions

  1. Connect your ESP32-S3 board to your computer via USB
  2. Click the "Install to ESP32-S3" button above
  3. Select the correct serial port when prompted
  4. Wait for the installation to complete
  5. Use the serial monitor or Improv WiFi to configure WiFi credentials
Note: This installer requires a Chrome, Edge, or Opera browser with Web Serial API support. Safari and Firefox are not currently supported.
WiFi Configuration: After flashing, you can configure WiFi using:
  • Serial monitor: Send WIFI:your-ssid,your-password
  • Or use SCAN to see available networks
  • Send STATUS to check connection status
